# Hospital Information System (HIS)

A Laravel-based Hospital Management System with modules:

- Patient Management
- Appointment System
- Doctor Management
- IPD / OPD
- Pharmacy Module
- Lab & Radiology
- Role & Permission (Spatie)
- Admin Dashboard

## Tech Stack
- Laravel 13
- MySQL
- AdminLTE
- Yajra DataTables
- Spatie Permission

## Features
- Role-based access control
- Token-based appointment system
- Dynamic scheduling
- Multi-role hospital workflow
